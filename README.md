## 🔍 Project Overview

This project analyzes a dataset of **50,000+ food ordering transactions** to uncover critical factors influencing consumer choices, delivery efficiency, and revenue generation. Using **Tableau**, the analysis translates raw transaction data into interactive, actionable visual insights.

### Key Focus Areas:
1. **Consumer Demographics & Behavior:** Understanding age-wise distributions, popular dining times, and meal preferences.
2. **Revenue & Ordering Trends:** Highlighting top-performing cities, average order values (AOV), and high-revenue cuisines.
3. **Logistics & Delivery Performance:** Analyzing delivery times across different cities to optimize supply-chain efficiency.
